# dockerize-bb-mockup

prerequisite:
you have installed docker-compose


clone this repo

```
git clone https://github.com/indraakkk/dockerize-bb-mockup.git
```

```
cd to dockerize-bb-mockup
```

run all container
```
docker-compose up -d
```

```
open mockup at localhost:80
bb at localhost:8004
```
